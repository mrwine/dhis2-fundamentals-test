Set Up - Consistency - Time graded exercise

Open the DHIS2 Data Quality training instance. Then, open the WHO Data Quality Tool from the applications menu.

Once the WHO Data Quality Tool has loaded, you can begin setting up the application to complete the Consistency-time exercise. Follow the instructions below:

  1. Click on the tab for the “Consistency - time” dashboard. With the menu open, click on the rectangle beneath “Data” and view the drop-down list. These are the various groups of indicators which can be reviewed with the dashboard of the WHO Data Quality Tool. Leave Data set to [Core].
  2. Set the period to January 2021.
  3. Select District from the Disaggregation drop-down menu.

Review the graph at the top left (ANC 1 Visits). Move the cursor over the lines and read the information in the box which pops up. Then, answer the 3 questions below.
